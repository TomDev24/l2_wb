Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
[3 2 3]

Слайс представляет собой структуру с такими полями как длина, емкость и указатель на массив. При передаче этой структуры в другую функцию она 
полностью копируется, но новая копия все равно указывает на тот же массив.

Внутри функции modifySlice(), аргумент i является копией структуры слайса, i[0] = 3 изменяет тот же низлежащий массив, что и у слайса s.
i = append(i, "4"), так как емкости слайса не хватает, чтобы добавить элемент, под капотом аллоцируется новый массив, с увеличенной памятью, куда и добавиться элмемент,
и теперь структура слайса i, имеет указатель на только что аллцированный массив и операция i[1] = 5, будет произведена над этим новым массивом и т.д..

Таким образом, в конце программы при выводе слайса s, мы получаем результат [3 2 3], так как i в modifySlice в самом начале еще указывал на тот же низлежащий массив
что и s.
```
